### Otherworld Guardian

This is a group project from a course at KTH. The adventure starts when you, playing as Beep the robot, spawn into the mystical and vibrant world of Celestria. As you arrive, you find out that you stand at the threshold of a realm shrouded in enchantment and peril. The once serene lands are now overshadowed by a creeping darkness, brought on by a malevolent force fueled by the Fungal Foes. The inhabitants of this world are in desperate need of salvation. The game combines world exploration with instances of puzzles.


### Pokémon Trivia

This is a group project from a course at KTH which is Pokemon quiz web application designed to be played on a computer. The Pokémon quiz lets users do quizzes to acquire Pokémons. The app uses the “PokéApi” to retrieve information about the Pokémon world to create a quiz for the user to do. If the user is not logged in, it is a normal quiz. If the user is logged in, high score is persisted and put on the leaderboard and the user can start to collect different Pokémons. The user can collect Pokémons by answering two questions correct in a row.
